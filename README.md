# Chris.now

Personal website built with Next.js.

Visit: **https://chris.now**

## Features

- Live clock with typewriter animations
- Character-by-character text reveals  
- Animated gradient background
- Discord handle clipboard copy

## Tech Stack

- Next.js 15 + React 19
- Tailwind CSS v4
- TypeScript

## Development

```bash
npm install
npm run dev
```
